# 배출 안내표 1,000개 필드 설명 v38

각 품목은 다음 기준으로 운영됩니다.

- `id`: 품목 고유번호
- `name`: 사용자에게 보이는 품목명
- `category`: 대분류
- `material`: 주 재질
- `recyclableStatus`: 재활용 가능성
- `riskLevel`: 위험도
- `disposalChannel`: 기본 배출 경로
- `householdMethod`: 가정 배출 방법
- `apartmentMethod`: 아파트·상가 배출 방법
- `businessMethod`: 사업장 배출 방법
- `agencyMethod`: 기관 물품 처리 방법
- `preparation`: 배출 준비
- `disposalMethod`: 상세 배출 방법
- `steps`: 기본 단계
- `disposalStepsDetailed`: 상세 단계
- `do`: 해야 할 것
- `dont`: 하면 안 되는 것
- `caution`: 주의사항
- `packaging`: 포장 기준
- `photoGuide`: 사진 촬영 기준
- `photoChecklist`: 사진 체크리스트
- `pickupRule`: 즉시수거 기준
- `instantPickup`: 즉시수거 후보 여부
- `minWeightKg`: 무게 기준
- `minValueKrw`: 예상가치 기준
- `countThreshold`: 수량 기준
- `feeDecisionGuide`: 요금/매입가 판단 기준
- `adminCheck`: 관리자 검토 조건
- `legalNote`: 법적·업무 확인사항
- `localDifference`: 지역별 차이
