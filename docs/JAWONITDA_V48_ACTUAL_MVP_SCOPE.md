# 자원잇다 v48 실제 적용형 MVP 범위

이번 버전은 화면 추가가 아니라 실제 운영 흐름을 기준으로 보강했다.

## 핵심 흐름
접수 → 위험검토 → 동별 묶음대기 → 입찰진행 → 낙찰 → 수거일정 → 현장검수 → 정산 → 리포트.

## 실제로 가능해진 것
- 회생 케이스 생성 API 및 화면
- 자동 위험검토와 S/A/B/C/D 등급 판정
- 동별 묶음 후보 자동 집계
- 업체 입찰 등록
- 낙찰, 상태변경, 정산 시뮬레이션
- 케이스별 회생 결과 리포트 생성
- 파트너 검증 점수화
- 30일 실행 태스크 관리

## 실영업 전 남은 것
- 사업자/세무 처리
- 계약서 최종 법률 검토
- 폐기물·자동차·데이터삭제 파트너 실제 계약
- 실제 결제·SMS·지도 API 키 입력
- 개인정보보호 점검

## 원칙
자원잇다는 초기에는 직접 수거·보관·해체·처리를 하지 않고, 검증된 파트너와 물량을 연결하는 지역운영사로 시작한다.
