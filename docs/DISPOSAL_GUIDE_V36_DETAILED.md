# 자원잇다 v36 상세 배출 안내표 운영 기준

## 목적
개인이 사진을 찍거나 품목명을 입력하면 단순 분리배출 안내가 아니라 실제 수거 운영에 필요한 판단까지 제공합니다.

## 고객에게 표시되는 정보
1. 품목명, 분류, 재질
2. 배출 경로: 분리배출 / 묶음수거 / 즉시수거 / 관리자 검토 / 전용수거
3. 가정·아파트·사업장·기관별 배출 방법
4. 단계별 배출 절차
5. 해야 할 것 / 하면 안 되는 것
6. 사진 촬영 기준
7. 즉시수거 기준
8. 묶음수거·캠페인 전환 기준
9. 요금·매입가 판단 기준
10. 법적·지역별 확인사항

## 관리자 판단 로직
- 즉시수거: 무게, 예상가치, 수량, 대형품 중 하나 이상 충족
- 묶음수거: 기준 미달이나 동일 지역 물량이 모일 가능성 있음
- 캠페인: PET, 박스, 헌옷 등 소액 다량 품목
- 관리자 검토: 위험품, 저장매체, 사업장폐기물, 건설폐기물, 냉매, 배터리, 대형 철거
- 수거불가/전용수거: 화학물질, 의료성 폐기물, 누액 배터리, 파손 형광등 등

## API
- GET /api/disposal/categories
- GET /api/disposal/items?q=&category=&limit=
- GET /api/disposal/items/{item_id}
- POST /api/disposal/guide
- PATCH /api/admin/disposal/items/{item_id}

## 사업화 포인트
배출 안내표는 무료 정보가 아니라 자원잇다의 운영 데이터입니다. 공개 화면에서는 검색 결과 일부와 안내 카드만 보여주고, 관리자와 유료 플랜에서는 전체 품목표·권한별 다운로드·API 연동을 제공합니다.
