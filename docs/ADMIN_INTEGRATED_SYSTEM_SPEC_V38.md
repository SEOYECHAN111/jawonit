# 관리자 통합시스템 상세 설계 v38

## 1. 관제판
- 전체 회원 수
- 심사 대기 수
- 수거신청 수
- 입찰방 수
- 제안서 수
- 문의 수
- 배출 안내표 수
- 보호자료 세트 수

## 2. 회원 심사
- 개인: 기본 승인
- 업체: 사업자등록증, 허가·신고자료, 차량등록증, 보험, 작업 가능 품목 확인
- 기관: 담당부서, 담당자, 공문 또는 위임자료 확인
- 기업: 사업자등록증, 사업장, ESG/총무/시설 담당자 확인

## 3. 배출 안내표 관리
- 1,000개 품목 검색
- 품목별 즉시수거 기준 확인
- 위험품/저장매체/건설폐기물/사업장폐기물 관리자 검토
- CSV 내보내기
- API 기준 수정
