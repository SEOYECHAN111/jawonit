폐자원회생

버려지는 물건을
회생 프로젝트로 바꿉니다

자원잇다는 생활폐기물을 무단 수거하는 회사가 아니라, 배출 전 자원가치·부품가치·재생가능성을 판단하고 허가·전문 파트너에게 연결하는 지역 운영사 플랫폼입니다.

1. 전자기기 회생

휴대폰·노트북·PC는 데이터삭제를 먼저 확인하고, 중고부품·도시광산·배터리처리로 분기합니다.

  • SSD/RAM/GPU/액정 분리 가치
  • 메인보드 금·은·팔라듐 회수
  • 전문 데이터삭제 확인서

2. 대형 폐가전 회생

냉장고·세탁기·에어컨은 공공 무상수거가 적합한지 먼저 보고, 가치가 있으면 파트너 회수로 연결합니다.

  • 압축기, 모터, 구리관, 알루미늄핀
  • 냉매·배터리·위험 요소 확인
  • 대형물류·허가 파트너 필요

3. 자동차 회생 프로젝트

아주 싼 차량이나 고장 차량을 자원잇다가 직접 해체하는 것이 아니라, 사전 검수 후 부품·재생·정산 프로젝트로 설계합니다.

  • 엔진·미션·전장품·휠·촉매 등 가치 분리
  • 자동차해체재활용업체·정비·재제조 파트너 연결
  • 판매금에서 비용·수수료를 제외하고 분배 구조 설계

4. 수집가치 회생

레트로 오디오, 카메라, 게임기, 희귀 부품은 일반 폐기물이 아니라 수집상·전문판매 경로로 연결합니다.

  • 사진 기반 희소성 검토
  • 전문 파트너 위탁판매
  • 낙찰·판매 후 정산
자동차 수익분배 예시

싼 차량 매입 → 파트너 검수 → 부품·재생 수익 분배

예: 저가 고장 차량을 바로 폐차하지 않고, 협력업체가 엔진·미션·휠·촉매·전장품을 평가합니다. 매각/재생 수익에서 매입비, 운송비, 검수비, 파트너 수수료를 제외한 금액을 프로젝트 기준에 따라 분배합니다.

① 매입 후보저가 차량, 사고차, 고장차, 장기 방치차
② 사전입찰파트너가 부품가치와 리스크를 먼저 견적
③ 합법 처리허가 업체·정비·재제조 파트너가 수행
④ 수익정산판매금 - 비용 - 수수료 = 분배 가능액

주의: 자원잇다가 직접 차량 해체·폐차·촉매 거래를 하는 구조가 아니라, 관련 허가와 자격을 가진 파트너가 수행하는 연결·정산 플랫폼 구조입니다.

배출 품목·방법

무조건 수거가 아니라, 맞는 배출경로를 안내합니다

폐휴대폰·노트북데이터삭제 확인 → 부품/중고/도시광산 경로
냉장고·세탁기·TV공공 무상수거 대상 여부 확인 → 대형가전 파트너
에어컨·실외기냉매·구리배관 확인 → 전문 철거/비철 파트너
배터리·보조배터리부풀음·누액 확인 → 폐배터리 전문 처리
가구·사무집기재사용 가능성 확인 → 중고가구/대형폐기물 안내
자동차 부품출처·소유·허가 확인 → 자동차 파트너 검토
