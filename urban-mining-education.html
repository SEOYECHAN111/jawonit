자원회생 운영모델

자원잇다는 폐기물이 아니라 회생경로를 관리합니다

모든 등록 물건은 먼저 수집가치, 중고가치, 부품가치, 재생가치, 원자재가치, 전문처리 필요성을 비교합니다.

최종 회생 판정 순서

  1. 수집가치가 있는지 먼저 확인
  2. 중고판매 가능한지 확인
  3. 부품판매·재제조 가능성 확인
  4. 원자재 가치와 kg 단가 확인
  5. 저장매체·위험품·출처불명 여부 확인
  6. 기준 미달은 동별 묶음대기로 전환

초기 금지 원칙

  • 생활폐기물 전체 수거 금지
  • 아파트 재활용장 무단 회수 금지
  • 무허가 폐기물 운반·보관 금지
  • 자동차 직접 해체 금지
  • 출처 불명 촉매·고가부품 거래 금지
  • 개인정보 저장매체 무단 재판매 금지

8개 카테고리

초기부터 넓게 받되, 실제 입찰은 최소 물량과 안전기준을 통과한 것만 진행합니다.

폐전자기기

PC, 노트북, 모니터, 서버, 휴대폰. 데이터삭제와 부품판매가 핵심.

사무 불용자산

학원, 병원, 사무실, PC방의 대량 물량. 패키지 수익.

아파트 캠페인

배출 전 소형가전·폐휴대폰·중고가구 QR 등록.

자동차 부품

정비소·폐차장·공업사 부품을 입찰로 회생.

노후차 프로젝트

A안 중개형, B안 직접매입 프로젝트형으로 운영.

엔진·미션 재생

코어 입찰, 재제조 위탁, 부품판매.

수집가치

레트로, 오디오, 카메라, 피규어, 희귀부품.

원자재·전문처리

전선, 알루미늄, 기판, 위험품, 보안폐기.
