# 자원잇다 v43 모바일 반응형 최종 점검

## 적용 내용
- 기존 이미지 로고 유지, 헤더 로고 크기 확대
- 980px 이하에서 모바일 햄버거 메뉴로 전환
- 메뉴 오픈 시 배경 dim 처리, ESC/외부 클릭/메뉴 클릭 시 닫힘
- 우측 하단 문의창을 모바일 바텀시트형으로 변경
- 문의창 접기/펼치기, 분류 선택, 관리자 문의목록 저장 확인
- 관리자 대시보드, 배출 안내표, API 연동센터, 입찰방, 수거신청 화면 모바일 1열화
- 표는 가로 스크롤 처리하여 화면 깨짐 방지
- 모든 입력창 font-size 16px 이상으로 모바일 확대/줌 문제 완화

## 실행 후 확인
- http://127.0.0.1:8000/?v=43
- Ctrl+F5 강력 새로고침
- 모바일 크롬 개발자도구에서 iPhone/Android 크기로 메뉴, 문의창, 로그인, 관리자 대시보드 확인
